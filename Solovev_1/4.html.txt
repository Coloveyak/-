<!-- 1. Разница между var и let
    let - не поднимается и имеет значение `undefined` только внутри блока, в котором объявлен
    let - имеет блочную область видимости (видны только внутри блока, в котором объявлены)
    let - не позволяет переобъявлять переменные

    var - позволяет переобъявлять переменные (создавать новую переменную с тем же именем)
    var - имеет функциональную область видимости (видны внутри всей функции)
    var - поднимается (инициализируется значением `undefined`) в начало функции или области видимости

2. Типы данных в Java Script
 Простые типы:
     Строка (string)
     Число (number)
     Булево значение (boolean)
     Null
     Undefined
 Сложные типы:
     Объект (object)
     Массив (array)
     Функция (function)
 Специальные типы:
     Символ (symbol)
     BigInt

3. Переменная, полученная с помощью команды `prompt`, сохраняется как string. -->
